# Тестовое задание

Тестовое задание на должность "Специалист технической поддержки 2 линии"


## Используемый стек

Фронтенд - React, библиотека интерфейсов Material UI.  
Бэкенд - FastAPI, Uvicorn, Docker.



## Deployment

Приложение развернуто на домашней виртуальной машине - доступ прокинут через маршрутизатор keenetic.
